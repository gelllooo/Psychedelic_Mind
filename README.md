# psychedelic__mind
A Flask web app to create and manage photo albums and posts. Users can add albums, create posts with titles, content, dates, and optional images, and view them on organized pages. Uploaded images are stored securely, with automatic folder creation to prevent errors. Perfect for learning Flask and file handling.

🚀 Features

* Create and manage blog posts with images
* Organize posts into albums (scrapbook-style collections)
* View individual posts with a clean journal layout
* Upload and display images in posts
* Minimalist scrapbook-inspired aesthetic


## 🛠️ Tech Stack

* Python (Flask)
* HTML/CSS
* SQLite

▶️ How to Run

1. Clone the repo
2. Install requirements:

   pip install -r requirements.txt
  
3. Run the app:


   python psychelic mind.py
   
4. Open in browser:

   
   http://127.0.0.1:5000/
